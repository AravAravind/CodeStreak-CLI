# CodeStreak CLI

A simple command-line application built with Python to track daily coding progress for the 100 Days of Code challenge.

---

## Features

- Add daily coding logs
- Store logs in a local file
- View all previous entries
- Displays total days completed
- Simple and easy-to-use CLI menu

---

## Tech Used

- Python 3
- File Handling
- datetime module

---

## Project Structure

CodeStreak-CLI/
│
├── tracker.py
├── logs.txt
└── README.md

---

## How to Run

1. Make sure Python 3 is installed.
2. Open terminal inside the project folder.
3. Run:

python tracker.py
